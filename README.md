### Running Locally

To develop locally, follow these steps:

1. **Clone this repo:**

    ```sh
    git clone https://github.com/javaistic/javaistic.git
    ```

2. **Navigate to the project root directory:**

    ```sh
    cd javaistic
    ```

3. **Install JS dependencies:**

    ```sh
    yarn
    ```

4. **Start the development server:**

    ```sh
    yarn dev
    ```
