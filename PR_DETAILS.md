This PR introduces the complete initial setup for **Javaistic**, a modern, comprehensive Java learning platform built with Next.js 15 and cutting-edge web technologies. This represents a major milestone in creating an accessible, interactive, and ad-free Java education resource.

## 📝 Description

This pull request establishes the foundation for Javaistic, a comprehensive Java learning platform designed to make Java programming accessible to developers of all skill levels. The platform features an extensive documentation system, practical program examples, and a modern, responsive user interface.

### What's Included

#### 🏗️ **Core Infrastructure**
- **Next.js 15** application with TypeScript support
- **Tailwind CSS** for modern, responsive styling
- **Fumadocs** integration for documentation management
- Comprehensive build and development tooling
- ESLint and Prettier configuration for code quality

#### 📚 **Content Structure**
- **32 Documentation Files** covering Java concepts from basics to advanced
  - Variables and Data Types
  - Control Flow (if-else, switch, loops)
  - Object-Oriented Programming
  - Arrays and Multidimensional Arrays
  - Exception Handling
  - Inheritance and Polymorphism
  - Static and Final Keywords
  - And much more...

- **22 Program Examples** with practical implementations
  - Basic I/O operations
  - Mathematical calculations
  - Array operations
  - String manipulation
  - Object-oriented examples
  - Algorithm implementations

#### 🎨 **User Interface Components**
- Modern, responsive design system
- Dark/Light mode support
- Advanced search functionality
- Interactive code examples
- Mobile-first responsive layout
- Accessibility-focused implementation

#### 🛠️ **Technical Features**
- **Performance Optimized**: Built with Next.js 15 for optimal loading times
- **SEO Ready**: Proper metadata and sitemap generation
- **Type Safe**: Full TypeScript implementation
- **Modern Fonts**: Inter, Funnel Display, and JetBrains Mono integration
- **Analytics Ready**: Google Analytics integration
